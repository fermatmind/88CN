# 88CN

88CN is the Free AI Project Growth Index.

Day 0 creates the project constitution, operating model, risk wall, and local verification scripts. It intentionally does not create an application framework, install dependencies, add deployment configuration, or create secrets.

## Phase 1 Scope

88CN Phase 1 is:

- a free AI project growth index
- a public growth-signal dashboard
- a founder claim workflow
- an editorial discovery layer
- a data foundation for future growth and interest products

88CN Phase 1 is not a generic directory, paid placement exchange, automatic publishing system, or private-data collection product.

## Local Checks

```bash
npm run docs:check
npm run policy:scan

```
