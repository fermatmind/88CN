# TRAFFIC2 Landscape Page Spec

Result: GO_TRAFFIC2B

## Scope

This is a future `/landscape` page architecture specification. It does not create routes, components, runtime code, sitemap code, or public assets.

## Hero

Purpose: establish 88CN as a reviewed AI project landscape and route users into safe discovery surfaces.

Approved message:

> Explore reviewed AI projects by category, stack, source links, and public readiness signals.

Hero should include:

- short positioning copy;
- one primary CTA to explore reviewed projects or current safe project list;
- one secondary CTA to run geo-checker or view reports;
- no unsupported counts;
- no paid placement;
- no broad superlative directory language.

## Audience Split

Future cards:

| Card | Audience | User job | Safe destination |
| --- | --- | --- | --- |
| For builders | Builders / repo hunters | Find active projects, stacks, source links, and alternatives. | Projects, stacks, collections, approved alternatives. |
| For founders | Potential AI founders | Understand sectors, competitors, and public readiness. | Landscape, reports, geo-checker, founders. |
| For researchers | Researchers / data buyers | Understand reviewed public signals and Alpha Feed evidence. | Reports, Alpha Feed evidence. |
| For project owners | AI project founders | Submit, claim, correct, or check source readiness. | Submit, claim/correction, geo-checker. |

## Browse By

Allowed cards:

- Stacks
- Collections
- Verticals
- Alternatives
- Reports
- Geo Checker
- Future Tasks
- Future Sectors

Rules:

- existing safe routes may be linked directly;
- future routes must be marked as coming soon or internal-only if not implemented;
- future routes must not be placed in sitemap;
- no fake pages may be generated.

## Public Signal Preview

Allowed examples:

- reviewed project count if source-backed;
- published project count;
- stacks available;
- collections available;
- verticals available;
- report count;
- last reviewed date where available.

Forbidden examples:

- unsupported global market count;
- projected total market size;
- unreviewed count;
- submitted/pending/quarantined/rejected counts as public coverage;
- any count affected by paid placement.

## Machine-Readability Evidence

Explain these signal types when source-backed:

- website reachable;
- GitHub linked;
- docs linked;
- sitemap detected;
- JSON-LD detected;
- canonical detected.

Do not imply that these signals guarantee ranking, citation, traffic, collection, exposure, customers, funding, or model answers.

## Founder CTA

Allowed CTAs:

- Submit project.
- Claim/correct public profile.
- Run geo-checker.

Rules:

- all submission/claim language must be human-reviewed;
- no automatic listing promise;
- no paid inclusion claim;
- no contact automation;
- no private data capture beyond existing approved workflows.

## Research / Alpha Feed CTA

Allowed CTAs:

- View reports.
- Read Alpha Feed evidence.

Rules:

- no live feed access;
- no customer signup;
- no buyer data capture;
- no private project fields;
- no implied commercial delivery in TRAFFIC2B.
