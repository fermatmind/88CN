# TRAFFIC1 Demand-Side Copy Boundary

Result: GO_TRAFFIC2

## Allowed Copy Patterns

Use these patterns when source-backed and truthful:

- reviewed public signals
- source-linked AI projects
- active source links
- machine-readability evidence
- public readiness signals
- reviewed sample count
- published project set
- source-backed open-source status
- source-backed commercial status
- last reviewed
- website reachable
- GitHub linked
- docs linked
- sitemap detected
- JSON-LD detected
- canonical detected
- founder not claimed
- public signals only
- not enough data
- source unavailable

## Forbidden Copy Patterns

Do not use these as public route promises:

- broad best/top/superiority claims without independent support
- ranking outcome promise
- guaranteed traffic
- guaranteed AI citation
- guaranteed ChatGPT visibility
- capital-access outcome promise
- guaranteed customers
- official endorsement without a source
- verified unless claim/verification status exists
- investment alpha
- financial advice
- capital-product and restricted financial-offering language covered by the AGENTS public-language ban
- fake count
- global market count without evidence
- copied competitor descriptions
- paid inclusion
- automatic listing
- dofollow/backlink promise
- exposure promise

## Chinese Forbidden Patterns

Avoid these phrases and claims in Chinese public copy:

- 保证曝光
- 保证排名
- 保证收录
- 保证被 ChatGPT 引用
- 免费外链
- 付费收录
- 买排名
- 最强
- 最全
- 第一
- 投资机会
- 收益
- 回报
- 股权
- 代币

If a future task needs to mention a risky term for policy or education, it must be clearly framed as a forbidden example or risk note, not as product copy.

## Route-Specific Copy Rules

| Route family | Allowed copy | Forbidden copy |
| --- | --- | --- |
| `/landscape` | Bilingual AI project landscape, reviewed public signals, source links, lifecycle status, machine-readability evidence. | Generic AI tool directory clone, unsupported market-size or tool-count claims. |
| `/landscape/sectors` | Reviewed sample counts, sector labels, source-backed project examples. | Global sector count claims, rankings, thin sector pages. |
| `/tasks/[slug]` | Projects for a concrete job, reviewed sample count, source-backed task fit. | Best AI tools for anything, unlimited task generation, copied descriptions. |
| `/alternatives/[slug]` | Public-signal comparison, source-backed differences, lifecycle status. | Better-than claims, reverse-pair duplicates, N-by-N pages. |
| `/zh-CN/landscape` | AI 项目图谱, 已审阅公开信号, 活体来源链接, 机器可读性证据. | AI工具导航 clone, 付费收录, 保证曝光. |
| `/zh-CN/geo-checker` | 官网检测, sitemap detected, JSON-LD detected, canonical detected, public readiness. | 促成收录, 保证被 ChatGPT 引用, visibility promise. |
| `/zh-CN/founders` | 出海项目官网准备, 公开信号, 来源链接, 人工审核边界. | Traffic, ranking, capital-access, or investment-style outcome promises. |
| `/zh-CN/submit` | Submit for human review, correction, claim, source-backed profile. | Automatic listing, paid inclusion, paid ranking, guaranteed collection. |
| `/report-distribution-pack` | Reviewed report package, source links, public evidence. | Automated outreach, private contact data, social posting automation. |

## Unknown Data Display

Unknown or unverified data must display safe labels such as:

- Not verified
- Public signals only
- Not enough data
- Founder not claimed
- Source unavailable

Do not fill gaps with estimates, invented counts, or competitor-derived copy.
