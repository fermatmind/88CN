- No Seed 100 project slug list.

- No staged import data.

- No new project pages.
- No project publishing action.
- No Supabase writes.
- No deployment action.
- No LLM call, paid external API call, browser automation, or headless JavaScript execution.

## Public-Copy Guardrails

The report copy is factual and aggregate-only. It avoids restricted public claims, external outcome promises, project promotion language, and paid placement framing. The report states that observed machine-readability signals are not promises about third-party systems or downstream business outcomes.

## Sitemap Policy

Only `/reports/early-ai-project-machine-readability-2026` is added to the sitemap. PR34 does not add Seed 100 project slugs, staged imports, quarantined imports, admin routes, API routes, or payment routes.

## Structured Data Policy

The page emits report-level `TechArticle` JSON-LD only. It includes 88CN as author and publisher, a report headline, description, publication date, and report topics. It does not include fake reviews, fake ratings, paid placement data, Signal Score values, Source Confidence values, or project-level item data.
