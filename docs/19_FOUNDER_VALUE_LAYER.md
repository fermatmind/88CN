# 19 Founder Value Layer

## Date

2026-06-15

## Purpose

Define the messaging, positioning, and content structure for founder-facing pages on 88CN. The founder value layer explains why AI project founders should create and claim profiles on 88CN, what value they receive, and — critically — what they should not expect.

## Pages

### /founders

Primary founder recruitment page. Explains the value proposition for AI project founders.

**Key messages:**

1. **Create a Clean, Structured Public Profile**
   A free AI project profile on 88CN provides a consistent, search-indexed public page with project name, description, category, public sources, and growth signal indicators.

2. **Improve Source Confidence**
   Claiming a project and linking verified public sources improves the source confidence level. Higher confidence means more reliable signal data.

3. **Add Public Project Details**
   Claimed projects can maintain up-to-date public information that is accurately represented in the index and discovered through category and collection pages.

4. **Prepare for Verified Signal Profile**
   Early claiming establishes presence in the index and positions the project for future signal enhancements, editorial inclusion, and report coverage.

5. **Optional Genesis Badge**
   Verified projects may be eligible for the Genesis Badge — an optional, informational recognition. No obligations, no ranking promises, no backlink requirements.

**Explicit disclaimers:**

- No traffic guarantees
- No ranking improvements
- No SEO benefits
- No backlink promises
- No guaranteed outcomes from listing or claiming
- No paid placement options
- No sponsored rankings
- No commercial obligations

### /genesis

Genesis Badge explainer page. Describes the badge concept, eligibility criteria, and limitations.

**Key messages:**

1. **What the Genesis Badge Represents**
   Recognition for early projects with established verifiable public growth signals and founder verification.

2. **Public Signals Only**
   Badge eligibility is based exclusively on publicly available data.

3. **Founder Claimed**
   Badge eligibility requires founder verification through supported methods.

4. **Source Confidence**
   Projects must maintain minimum source confidence levels.

5. **Signal Profile**
   The badge reflects a project's signal profile across all six dimensions.

6. **Optional Embed Planned**
   Future optional embed for project websites. No backlink requirements.

7. **No Forced Backlinks**
   The badge does not require, imply, or incentivize backlinks.

8. **No Ranking Guarantees**
   The badge is not a ranking, endorsement, or commercial certification.

## Tone and Language Rules

### Must Use

- Free AI project profile
- Public project page
- Editorial listing
- Search-indexed profile
- Founder claim
- Public growth signals
- Signal Score
- Source confidence
- Growth readiness
- Subscribe to updates
- Express interest
- Join waitlist

### Must Not Use

- dofollow backlink
- 301 permanent backlink
- SEO juice
- guaranteed ranking
- invest now
- guaranteed return
- guaranteed yield
- guaranteed profit
- principal protection
- 保本, 稳赚, 固定收益, 回购保障
- SPV, fund, AI fund, equity, token, ICO, IDO, securities
- buy equity, deposit funds, real-money commitment
- fake backer count, fake watchlist count
- fake MRR, fake ARR, fake user count
- fake customer logos

## Founder Journey

The planned founder journey (not yet implemented):

1. **Discovery** — Founder discovers their project on 88CN
2. **Submit** — If not yet indexed, founder submits project for editorial review
3. **Claim** — Founder claims existing profile through verification (domain email, DNS TXT, GitHub, manual review)
4. **Verify** — 88CN editorial team reviews and verifies claim
5. **Manage** — Founder updates public project information
6. **Track** — Founder monitors Signal Score and growth signals over time

## Non-Goals

The founder value layer does NOT:

- Provide a founder dashboard or admin panel (future phase)
- Enable real-time profile editing (future phase)
- Send email notifications (future phase)
- Offer paid services or premium profiles
- Provide traffic analytics or SEO tools
- Guarantee any outcome from listing or claiming
