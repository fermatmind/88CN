# Prohibited Tools

The following tool categories are denied by default.

## Cookie Readers

Tools that read browser cookies can expose user sessions.

## Password And Keychain Readers

Tools that read saved passwords or system credential stores are out of scope.

## Unrestricted Filesystem Tools

Tools with full disk access can expose private files outside the repo.

## SEO Rank Or Link Automation Tools

Rank tracking, backlink automation, and link-placement automation can create misleading public claims and are outside Phase 1.

## Auto Social Posting Or DM Blast Tools

Automated public posting or direct-message tools can send unreviewed outbound communication.

## Cloud Write Plugins

Cloud write plugins can mutate infrastructure and require human approval.

## Live Payment Tools

Live payment tools can create real-money side effects and are human-owned.

## Production Database Write Plugins

Production database write plugins can mutate live records and require explicit human approval outside Codex.

## Secret-Scanning Bypass Tools

Tools that hide or bypass credential scanning are forbidden.

## Browser Extension Installers

Browser extensions may read profile state and are denied by default.
